## Summary

OVHcloud Public Cloud `loadbalancer`: Manage loadbalancers in the given cloud project

## Child Command Groups

- `ovhcloud cloud`: Manage your projects and services in the Public Cloud universe (MKS, MPR, MRS, Object Storage...)
- `ovhcloud cloud loadbalancer associate-floating-ip`: Associate an existing floating IP to a loadbalancer
- `ovhcloud cloud loadbalancer create`: Create a loadbalancer in the given cloud project
- `ovhcloud cloud loadbalancer create-floating-ip`: Create a floating IP and attach it to a loadbalancer
- `ovhcloud cloud loadbalancer delete`: Delete a specific loadbalancer
- `ovhcloud cloud loadbalancer edit`: Edit the given loadbalancer
- `ovhcloud cloud loadbalancer get`: Get a specific loadbalancer
- `ovhcloud cloud loadbalancer health-monitor`: Manage health monitors of loadbalancers
- `ovhcloud cloud loadbalancer l7policy`: Manage L7 policies of loadbalancers
- `ovhcloud cloud loadbalancer list`: List your loadbalancers
- `ovhcloud cloud loadbalancer listener`: Manage listeners of loadbalancers
- `ovhcloud cloud loadbalancer log`: Manage loadbalancer logs
- `ovhcloud cloud loadbalancer pool`: Manage pools of loadbalancers
- `ovhcloud cloud loadbalancer stats`: Get statistics for a loadbalancer
