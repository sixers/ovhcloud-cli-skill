## Summary

OVHcloud Public Cloud `kube`: Manage Kubernetes clusters in the given cloud project

## Child Command Groups

- `ovhcloud cloud`: Manage your projects and services in the Public Cloud universe (MKS, MPR, MRS, Object Storage...)
- `ovhcloud cloud kube create`: Create a new Kubernetes cluster
- `ovhcloud cloud kube customization`: Manage Kubernetes cluster customizations
- `ovhcloud cloud kube delete`: Delete the given Kubernetes cluster
- `ovhcloud cloud kube edit`: Edit the given Kubernetes cluster
- `ovhcloud cloud kube get`: Get the given Kubernetes cluster
- `ovhcloud cloud kube ip-restrictions`: Manage IP restrictions for Kubernetes clusters
- `ovhcloud cloud kube kubeconfig`: Manage the kubeconfig for the given Kubernetes cluster
- `ovhcloud cloud kube list`: List your Kubernetes clusters
- `ovhcloud cloud kube node`: Manage Kubernetes nodes
- `ovhcloud cloud kube nodepool`: Manage Kubernetes node pools
- `ovhcloud cloud kube oidc`: Manage OpenID Connect (OIDC) integration for Kubernetes clusters
- `ovhcloud cloud kube private-network-configuration`: Manage private network configuration for Kubernetes clusters
- `ovhcloud cloud kube reset`: Reset the given Kubernetes cluster
- `ovhcloud cloud kube restart`: Restart control plane apiserver to invalidate cache without downtime
- `ovhcloud cloud kube set-load-balancers-subnet`: Update the load balancers subnet ID for the given Kubernetes cluster
- `ovhcloud cloud kube update`: Update the given Kubernetes cluster
