## Summary

OVHcloud Public Cloud `storage`: Manage storage services in the given cloud project

## Child Command Groups

- `ovhcloud cloud`: Manage your projects and services in the Public Cloud universe (MKS, MPR, MRS, Object Storage...)
- `ovhcloud cloud storage block`: Manage block storage volumes in the given cloud project
- `ovhcloud cloud storage file`: Manage file storage shares in the given cloud project
- `ovhcloud cloud storage object`: Manage S3™* compatible storage containers in the given cloud project (* S3 is a trademark filed by Amazon Technologies,Inc. OVHcloud's service is not sponsored by, endorsed by, or otherwise affiliated with Amazon Technologies,Inc.)
- `ovhcloud cloud storage swift`: Manage SWIFT storage containers in the given cloud project
